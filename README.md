# DNA methylation profile of _Xylanibacter ruminicola_ changes across seasons and host pregnancy

This is a study to identify the genome-wide DNA methylation variation in _Xylanibacter ruminicola_ in bovine rumen across differetn host preganacy status and seasons.
The DNA methylation was characterized by Oxford Nanopore sequencing.
